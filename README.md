# Ping Pong Game 🎮

## Overview
A classic single-player Ping Pong game built with **C++** and **Raylib**.  
The player controls a paddle to bounce the ball and score points. The game gets faster as you score more!

## Features
- 🟢 Paddle movement with arrow keys  
- 🔴 Ball collision and rebound mechanics  
- 🟡 Increasing speed after every 5 points  
- ⚪ Highscore saving and loading (stored in `Highscores.txt`)  
- 🎨 Simple menu with Play, Highscores, and Exit options  

## Controls
- **Up Arrow**: Move paddle up  
- **Down Arrow**: Move paddle down  
- **Enter**: Select menu option / confirm name  
- **Escape**: Exit game or cancel input
